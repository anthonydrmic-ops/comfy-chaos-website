# Comfy Chaos — Website

Marketing site for **Comfy Chaos Pty Ltd** — a software development studio.
*Clarity in complexity.*

Single-page static site (`index.html`) with the logo assets in `assets/`.
Hosted on GitHub Pages.

## Updating the site
1. Edit `index.html` (or assets).
2. Commit and push:
   ```bash
   git add -A
   git commit -m "your change"
   git push
   ```
Pushing to `main` automatically redeploys GitHub Pages within a minute.
